# Lista de Tareas

Una pagina web para gestionar tareas pendientes. Este proyecto permite añadir, marcar como completadas y eliminar tareas, utilizando almacenamiento local para que los datos se mantengan visibles incluso despues de cerrar o refrescar el anvegador.

## Caracteristicas

* **Guardado de datos:** Utiliza 'localStorage' para guardar las tareas de forma local.
* **Diseño minimalista:** Diseño limpio y adaptable, centrado en la pantalla.
* **Interacciones Animadas:** Efectos de hover, transiciones suaves y una animacion en CSS al eliminar elementos.
* **Agregado rapido:** Permite agregar tareas rapidamente presionando la tecla 'Enter'.

## Tecnologias Utilizadas

* **HTML:** Estructura del documento.
* **CSS:** Estilos personalizados, transiciones y animaciones.
* **JavaScript:** Manipulacion del DOM, manejo de eventos y gestion de estado.
* **Google Fonts:** Fuente *Lato*..
